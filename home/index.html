What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for interacting with HTML, XML, and other structured documents. It represents the document as a tree structure, where each node represents an element, attribute, or text content. This tree structure allows developers to access and manipulate the document's content, structure, and style.

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation in the context of the DOM is a technique where you attach an event listener to a parent element instead of individual child elements. This allows you to handle events for all child elements efficiently, especially when there are many dynamic elements or elements that are added or removed from the DOM.

How do you manipulate an element's attributes and styles using the DOM?

Attributes and styles are essential components of HTML elements, defining their properties and appearance. The DOM provides methods and properties to access and modify these attributes and styles.
